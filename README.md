# api-node-solid
 API de exemplo desenvolvida com Node.js, TypeScript e Express, seguindo os princípios da Clean Architecture e SOLID. O projeto implementa um fluxo completo de criação de usuário, incluindo envio de e-mail de boas-vindas, separação de responsabilidades por camadas (entidades, repositórios, provedores, casos de uso e controllers) e boas práticas de organização e extensibilidade.
